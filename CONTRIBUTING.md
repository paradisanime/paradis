This repository accepts contributions that improve **documentation**,  
**issue clarity**, and **release hygiene**. It does not accept private
application source code.

---

## Welcome Contributions

```
  + Fix typos / broken links in docs
  + Improve install / FAQ wording based on real user friction
  + Add screenshots to assets/ (with personal data scrubbed)
  + Improve Issue templates
```

## Not Accepted Here

```
  x App source code pull requests
  x Keystores, secrets, tokens
  x Unsigned debug APKs claimed as production
  x External binary re-uploads from unknown origins
```

---

## Doc PR Checklist

```
  [ ] ASCII / symbol decor only (no emoji decoration filler)
  [ ] No hardcoding of secrets
  [ ] Links still resolve
  [ ] Package id remains documented as com.paradis.app
  [ ] version.json schema unchanged unless intentional
```

---

## Issue First

Large documentation restructures should open an Issue first so maintainers
can agree on structure before a large PR lands.
