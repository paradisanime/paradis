# Brand & Screenshot Assets

Place public marketing images here.

```
  brand/
    icon.png
    banner.png

  screenshots/
    home.png
    detail.png
    player.png
    search.png
```

Guidelines:

```
  . Prefer lossless or high-quality PNG / WebP
  . Avoid watermark spam
  . Do not include personal account tokens visible in screenshots
  . Keep filenames lowercase-kebab-case
```
